# 🏦 Console 팀프로젝트 - 은행 서비스 **BB 뱅크**

> **JAVA 콘솔**을 이용한 *편의성과 보안을 강화한 은행 서비스*  
> 개발 기간: **2025.07.28 ~ 2025.08.08**

---

## 📌 프로젝트 개요
**BB 뱅크**는 콘솔 환경에서 회원가입, 계좌 관리, 송금, 입출금 등 주요 은행 기능을 수행할 수 있는 서비스입니다.  
JAVA 언어를 기반으로 하여 **보안성(OTP, 이메일 인증)**과 **사용자 편의성(비밀번호 찾기, 계좌 관리 자동화)**을 강화하였습니다.

---

## 🎯 개발 목표
> JAVA 콘솔을 이용한 **편의성을 해치지 않는 보안 강화 은행 서비스**

- 회원 인증 및 계정 관리
- 이메일 인증 기반 OTP 시스템 구현
- 입출금 및 송금 내역 관리
- 100만원 이상 송금 시 추가 본인확인(이중 보안)
- 콘솔 UI 기반 직관적인 사용자 흐름 설계

---

## 🖥 주요 기능

### 1️⃣ 메인 화면
- 프로그램 실행 시 로그인 및 회원 관리 메뉴 출력  
- 신규 회원가입 / 로그인 / 종료 기능 제공  

### 2️⃣ 계정 관리
- 회원가입, 아이디 찾기, 비밀번호 찾기 등 계정 관련 기능 수행  
- 회원가입 시 이메일 인증을 통해 **6자리 난수 OTP** 발급  
- OTP를 통해 추가 보안 절차 수행  

### 3️⃣ 계좌 관리
- 계좌 개설, 해지, 이체 기능 제공  
- 여러 계좌를 동시에 관리 가능  
- 계좌별 입출금 내역 조회 가능  

### 4️⃣ 송금 및 보안 기능
- 송금 시 금액에 따라 **OTP 인증** 추가  
- 100만원 이상 송금 시 **추가 본인확인 절차** 수행  
- 보안 강화를 위한 이중 인증 절차 적용  

---

## ⚙️ 구현 방식
- **Java Console 기반 입출력 처리**
- **Random**를 활용한 OTP, 비밀번호 검증
- **객체지향 설계(OOP)**로 회원, 계좌, 보안 모듈 분리
